# 🛒 ShopSphere - PHP E-Commerce Website

A full-stack E-Commerce web application built using **PHP**, **MySQL**, **HTML**, **CSS**, **Bootstrap**, and **JavaScript**. The project provides a complete online shopping experience with separate user and admin panels.

---

## 📌 Features

### 👤 User Features

- User Registration & Login
- Secure Password Hashing
- Browse Products
- Search Products
- Category Filter
- Price Sorting
- Product Details Page
- Shopping Cart
- Wishlist
- Shipping Address
- Cash on Delivery (COD)
- My Orders
- Order Invoice
- User Profile Management
- Product Reviews & Ratings
- Responsive Design

---

### 🔑 Admin Features

- Secure Admin Login
- Dashboard
- Manage Products
- Add/Edit/Delete Products
- Manage Categories
- Manage Orders
- Update Order Status
- View Customer Orders
- Dashboard Statistics

## ⚙️ Installation

1. Install XAMPP.
2. Copy the project into:

```
htdocs/ecommerce
```

3. Start:

- Apache
- MySQL

4. Open phpMyAdmin.

5. Create a database:

```
ecommerce_db
```

6. Import the SQL file.

7. Open your browser:

```
http://localhost/ecommerce
```
